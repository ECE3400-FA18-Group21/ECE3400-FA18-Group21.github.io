Milestone 3

Milestone Goals

Implement a maze navigation algorithm that effectively explores arbitrary mazes.

Navigation Algorithm- DFS

  • We chose to implement a modified DFS algorithm because it is relatively simple to get working and does not use very much memory.
  • Our DFS prioritizes moving straight over turning left or right, as turning takes time.
  • The robot will continue moving to unexplored grid spaces until it either hits a dead end or encounters only grid spaces it has visited before.
  • When the robot encounters a dead end (3 walls) it begins reversing its movement.
  • Unless hte robot has explored everything, there will be some unexplored node along the robot's path. It will continue retracing its steps until it finds an unexplored grid space, at which point it will turn and explore the new area.
  • To support this, we store the coordinates of each grid node, so that the robot can retrace its steps in the opposite direction. This is stored in an array that functions as a stack (with a pointer that we change to "pop" coordinates off of the stack).
  • When the robot finds that it can only see coordinates that it has encountered before, we move into one of these spaces. We then search in our traversal stack for the coordinate we just moved into, and pop off everything that came after it. We then continue reversing our steps as described above.
  • Below is pseudo-code demonstrating the algorithm.

      void dfs(){
        if (forward_unexplored){
          move_forward();
          push_coordinate_stack();
        }
        else if (right_unexplored){
          turn_right();
          move_forward();
          push_coordinate_stack();
        }
        else if (left_unexplored){
          turn_left();
          move_forward();
          push_coordinate_stack();
        }
        else if (only_see_walls){
          turn_around();
          move_forward();
          retrace_steps();
        }
        else{
          move_visited_grid();
          pop_coordinates_stack();
          retrace_steps();
        }
      }

      void retrace_steps(){
        pos = pop_coordinate_stack();
        move_to_coordinate(pos);
        if (see_unexplored)
          dfs();
        else
          retrace_steps();
      }
